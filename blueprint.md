# 프로젝트 블루프린트: 모던 스도쿠 게임 (Modern Sudoku Game)

## 1. 개요 (Overview)
이 프로젝트는 HTML, CSS, JavaScript만을 사용하여 구축된 현대적이고 반응형인 웹 기반 스도쿠 게임입니다. Firebase Studio 환경에서 실행되며, 깔끔한 UI/UX와 부드러운 인터랙션을 목표로 합니다.

## 2. 프로젝트 상세 명세 (Project Specifications)

### 2.1. 디자인 & 스타일 (Design & Style)
*   **테마:** 부드러운 그림자와 라운딩 처리가 된 "Glassmorphism" 또는 깔끔한 "Flat Design" 스타일.
*   **색상 팔레트:**
    *   배경: 은은한 뉴트럴 톤 (예: `#f0f2f5`).
    *   보드: 흰색 배경에 깔끔한 그리드 라인.
    *   강조색: 활기찬 블루 또는 퍼플 계열 (선택된 셀, 숫자 입력 등).
    *   오류: 부드러운 붉은색.
*   **레이아웃:**
    *   중앙 정렬된 게임 보드.
    *   보드 아래 또는 옆에 컨트롤 패널 (새 게임, 난이도 등).
    *   모바일 친화적인 반응형 디자인.

### 2.2. 기능 (Features)
*   **게임 로직:**
    *   무작위 스도쿠 퍼즐 생성 (Backtracking 알고리즘 사용).
    *   난이도 선택 (빈 칸의 개수 조절).
    *   유효성 검사 (가로, 세로, 3x3 박스 중복 체크).
*   **인터페이스:**
    *   9x9 그리드 클릭/터치 하여 셀 선택.
    *   키보드 숫자 키 또는 화면의 숫자 패드(Numpad)로 입력.
    *   실시간 충돌 감지 (같은 행/열/박스에 같은 숫자가 있을 경우 시각적 알림).
    *   "새 게임" 버튼.

### 2.3. 기술 스택 (Tech Stack)
*   **HTML:** 시맨틱 태그 사용 (`main`, `section`, `nav`).
*   **CSS:** CSS Grid (보드 레이아웃), Flexbox (UI 배치), CSS Variables (테마 관리).
*   **JavaScript:** ES Modules, Class 기반 구조 (`SudokuGame`, `SudokuUI`).

## 3. 현재 작업 계획 (Current Action Plan)
1.  **`index.html` 작성:**
    *   게임 컨테이너, 헤더, 보드 영역, 컨트롤 영역 구조화.
    *   Google Fonts (예: 'Poppins' or 'Roboto') 연동.
2.  **`style.css` 작성:**
    *   CSS 변수 설정.
    *   9x9 그리드 스타일링 (3x3 구획 강조).
    *   반응형 미디어 쿼리 추가.
    *   애니메이션 효과 (숫자 입력 시, 클리어 시).
3.  **`main.js` 작성:**
    *   스도쿠 생성 알고리즘 구현.
    *   UI 렌더링 및 이벤트 리스너 (클릭, 키보드 입력) 처리.
    *   게임 승리 조건 체크 로직.
